# Figma Components

Liệt kê tất cả components được định nghĩa trong một Figma file.

## Instructions

Gọi tool `get_file_components` với `file_key`.

Hiển thị danh sách components kèm tên, description, key để có thể dùng lại.
Nếu user muốn, có thể gợi ý code React component tương ứng dựa trên tên component Figma.

## Output format

```
🧱 Components trong file <file_key> (<n> components):

  [<key>] <name>
  Description: <description>
  ---
```

## Example

`/figma-components MO4JcMsNudV8vtIwmCPGoc`
